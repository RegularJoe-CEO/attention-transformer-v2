# Clone on RunPod (3 commands)

Repo: https://github.com/RegularJoe-CEO/attention-transformer-v2

In the **RunPod web terminal** (`root@...`), paste:

```bash
cd /workspace
git clone https://github.com/RegularJoe-CEO/attention-transformer-v2.git
cd attention-transformer-v2
bash scripts/runpod_v2_test.sh
```

No token, no Mac Desktop path, no `pod-setup`.
